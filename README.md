# 📊 Purchase Pattern Analytics

## 📌 Project Overview

This project analyzes customer purchase behavior using **Market Basket Analysis** to identify products that are frequently bought together. The Apriori algorithm was used to generate association rules, while Power BI dashboards were developed to visualize sales patterns and support business decision-making.

---

## 🎯 Business Problem

Retail businesses need to understand customer purchasing behavior to:
- Increase cross-selling opportunities
- Improve product placement
- Create effective promotional bundles
- Enhance customer shopping experience

This project identifies hidden relationships between products using association rule mining.

---

## 📂 Dataset

- Online Retail Transaction Dataset
- More than **510,000 retail transactions**
- Features include:
  - Invoice Number
  - Product Description
  - Quantity
  - Unit Price
  - Customer ID
  - Country
  - Invoice Date
